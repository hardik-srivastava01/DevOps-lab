## Issues Faced

Initially, I was confused about the difference between **Archive** and **Compression**. After practicing, I understood that:

- `tar` combines multiple files into a single archive but does not reduce file size.
- `gzip` compresses a file to reduce its size.
- `.tar.gz` is created by first archiving files and then compressing the archive.
- `zip` performs archiving and compression together.
- `du` shows the size of files/directories, while `df` shows available and used disk space of the filesystem.

I also learned how to verify archive contents before extraction and when to use each command in real-world scenarios.
